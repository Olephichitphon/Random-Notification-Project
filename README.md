This project implements a Random Notification system that displays various messages at the corner of the screen using JavaScript.

### Key Learnings:
* **Dynamic DOM:** Creating and removing notification elements on the fly.
* **Random Logic:** Using `Math.random()` to pick messages from a predefined array.
* **Timing Functions:** Controlling how long notifications appear using `setTimeout`.
* **CSS Transitions:** Creating smooth animations for notification entry and exit.
